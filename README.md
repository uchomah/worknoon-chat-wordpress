# Worknoon Chat - WordPress Plugin

WordPress integration for real-time chat system with WooCommerce support.

## 🚀 Technologies

- **PHP 7.4+**
- **WordPress 5.0+**
- **REST API**
- **AJAX**
- **WebSocket**

## 📋 Features

- ✅ Custom Post Type for chat sessions
- ✅ Shortcode support `[worknoon_chat]`
- ✅ REST API endpoints
- ✅ AJAX messaging
- ✅ WooCommerce compatible
- ✅ Floating chat widget
- ✅ Product-based chat context (bonus)

## 🔧 Installation

### Method 1: Upload via WordPress Admin

1. Download the plugin zip file
2. Go to WordPress Admin → Plugins → Add New
3. Click "Upload Plugin"
4. Select the zip file
5. Click "Install Now" → "Activate"

### Method 2: Manual Installation

1. Upload `worknoon-chat-wordpress` folder to `/wp-content/plugins/`
2. Activate plugin through 'Plugins' menu
3. Add shortcode to any page
